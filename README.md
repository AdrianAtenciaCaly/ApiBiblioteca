Prueba técnica realizada a la empresa SMDIGITAL, para aspirante a puesto de programado Web
Nota: para este desarrollo las tecnologías utilizadas fueron 
VuJS
.NetCore 5.
SQL Server.
Ide ututilizados:
Visual estudio code.
Visual studio 2019.
SQL Manager.
Paso a realizar para ejecutar la aplicación:
1: se debe generar la base de datos a partir de script suministrado (BIBLIOTECA.SQL), este se importa en el gestor de base datos.
2. abrir el proyecto del api con visual studio 2019 y ejecutar.
3. abrir el proyecto de biblioteca con visual studio o en su defecto abrir una consola (terminal) dentro de la carpeta del proyecto y ejecutar el comando npm run server. Posteriormente la url mostrada se debe copiar y pegar en el navegador de preferencia. 
Nota : debe tener instalado nodeJS
